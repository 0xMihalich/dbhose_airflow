CHUNK_SIZE
==========

.. py:data:: CHUNK_SIZE

   :type: int
   :value: 1048576

   Стандартный размер чанка для потоковой обработки данных.

**Описание:**

Константа определяет оптимальный размер блока данных (1 МБ = 1024 × 1024 байт)
для потоковой обработки при чтении и записи файлов. Используется для баланса
между производительностью и потреблением памяти.

**Использование:**

- Чтение больших файлов порциями (``file_writer``)
- Запись данных блоками в Native формат
- Буферизация сетевых передач
- Оптимизация операций ввода/вывода

**Преимущества:**

- **Производительность:** Оптимальный размер для большинства файловых систем
- **Память:** Фиксированное потребление независимо от размера данных
- **Совместимость:** Работает с любыми объемами данных

**Примечание:**

Соответствует рекомендациям для эффективной работы
с ClickHouse Native форматом и бинарными форматами баз данных.

CompressionMethod
=================

.. py:class:: CompressionMethod

   Перечисление поддерживаемых методов сжатия.

**Описание:**

Определяет кодеки сжатия, используемые для передачи данных в ClickHouse.
Каждый метод имеет числовой идентификатор и строковое представление.

**Значения:**

.. py:data:: NONE
   :value: 0x02

   Без сжатия.

.. py:data:: LZ4
   :value: 0x82

   LZ4 алгоритм (быстрое сжатие).

.. py:data:: ZSTD
   :value: 0x90

   ZSTD алгоритм (хороший баланс скорости и сжатия).

**Свойства:**

.. py:attribute:: method

   :type: str

   Строковое имя метода в нижнем регистре (например, ``"zstd"``).

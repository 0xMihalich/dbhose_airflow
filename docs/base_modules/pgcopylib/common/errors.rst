errors
======

Модуль исключений для работы с бинарным форматом PostgreSQL COPY.

PGCopySignatureError
--------------------

.. py:exception:: PGCopySignatureError

   **Родитель:** ``ValueError``

   Ошибка сигнатуры бинарного формата PostgreSQL COPY.

   **Когда возникает:**
   - Файл не содержит корректной сигнатуры ``PGCOPY\n\xff\r\n\0``
   - Формат файла не соответствует бинарному COPY формату
   - Поврежденный или неполный заголовок файла

   **Пример:**
   - ``Signature not match``

PGCopyRecordError
-----------------

.. py:exception:: PGCopyRecordError

   **Родитель:** ``ValueError``

   Ошибка структуры записей в бинарном формате.

   **Когда возникает:**
   - Несоответствие длины записи ожидаемому размеру
   - Поврежденные данные в середине потока
   - Некорректное количество колонок в записи
   - Преждевременный конец файла

   **Пример:**
   - ``Record length error``

**Использование:**

Для обработки ошибок при парсинге бинарных данных PostgreSQL/Greenplum.

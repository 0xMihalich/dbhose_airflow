DEFAULT_BLOCK_SIZE
===================

.. py:data:: DEFAULT_BLOCK_SIZE

   :type: int
   :value: 1048576

   Размер блока по умолчанию для Native формата ClickHouse.

**Описание:**

Константа определяет стандартный размер блока данных (1 МБ = 1024 × 1024 байт),
который используется при чтении и записи данных в Native формате ClickHouse.

**Использование:**

- Разбиение больших объемов данных на блоки
- Оптимизация передачи между клиентом и сервером
- Управление памятью при обработке потоков

**Примечание:**

Соответствует рекомендациям ClickHouse для эффективной работы с Native форматом.

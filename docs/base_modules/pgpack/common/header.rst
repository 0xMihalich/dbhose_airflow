HEADER
======

.. py:data:: HEADER

   :type: bytes
   :value: ``b"PGPACK\n\x00"``

   Сигнатура формата PGPack.

**Описание:**

Магическое число, идентифицирующее файлы в формате PGPack.
Состоит из строки "PGPACK" с переводом строки и нулевым байтом в конце.

**Использование:**

- Проверка корректности файла при открытии
- Запись в начало файла при создании
- Валидация формата в ``PGPackReader``

**Размер:**

8 байт (7 символов + нулевой байт).

**Примечание:**

Любой файл PGPack должен начинаться с этой сигнатуры.

PGPackDumperError
=================

.. py:exception:: PGPackDumperError

   **Родитель:** ``Exception``

   Базовое исключение для ошибок PGPackDumper.

**Когда возникает:**

- Общие ошибки инициализации PGPackDumper
- Проблемы с подключением к базе данных
- Неожиданные системные ошибки

**Примеры:**

- Ошибки подключения к PostgreSQL/GreenPlum
- Проблемы с аутентификацией

PGPackDumperWriteBetweenError
=============================

.. py:exception:: PGPackDumperWriteBetweenError

   **Родитель:**

   ``PGPackDumperWriteError``

   Ошибка переноса данных между источниками в PGPackDumper.

**Когда возникает:**

- Проблемы при выполнении ``write_between()``
- Ошибки соединения между серверами
- Несовместимость структур таблиц источника и назначения
- Проблемы с правами доступа на удаленном сервере

**Примеры:**

- Ошибки сетевого соединения между серверами
- Отсутствие таблицы на удаленном сервере

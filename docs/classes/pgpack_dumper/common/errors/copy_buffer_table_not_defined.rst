CopyBufferTableNotDefined
=========================

.. py:exception:: CopyBufferTableNotDefined

   **Родитель:** ``ValueError``

   Таблица назначения не определена.

**Когда возникает:**

- Не указаны ни ``query``, ни ``table_name`` в CopyBuffer
- Переданы пустые параметры для операций COPY

**Примеры:**

- ``Query or table not defined.``
- ``Table not defined.``

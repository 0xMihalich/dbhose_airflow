random_name
===========

.. py:function:: random_name()

   :return: Случайное имя для временных объектов
   :rtype: str

**Описание:**

Генерирует уникальное имя для подготовленных запросов и временных таблиц.

**Формат имени:**

``session_<16_hex_digits>`` (например: ``session_1a2b3c4d5e6f7890``)

**Использование:**

- Имена подготовленных запросов (prepare)
- Временные таблицы (temp table)
- Сессионные объекты

transfer_diagram
================

.. py:function:: transfer_diagram(source, destination)

   :param source: Метаданные источника
   :type source: DBMetadata
   :param destination: Метаданные назначения
   :type destination: DBMetadata
   :return: ASCII диаграмма передачи данных
   :rtype: str

   Генерирует текстовую диаграмму для визуализации передачи данных.

**Описание:**

Создает ASCII-арт диаграмму, показывающую структуру таблиц источника и назначения
с стрелкой передачи данных между ними. Используется для логирования операций
переноса данных.

**Пример вывода:**

.. code-block:: text

    Transfer data diagram:
    ┌───────────────────────────────────────────┐         ┌───────────────────────────────────────────┐
    │ Source [python iterable object]           │  │╲     │ Destination [clickhouse 23.8.1.2473]      │
    ╞═══════════════════╤═══════════════════════╡  │ ╲    ╞═══════════════════╤═══════════════════════╡
    │ Column Name       │ Data Type             │ ┌┘  ╲   │ Column Name       │ Data Type             │
    ╞═══════════════════╪═══════════════════════╡ │    ╲  ╞═══════════════════╪═══════════════════════╡
    │ id                │ UInt64                │ │     ╲ │ id                │ UInt64                │
    ├───────────────────┼───────────────────────┤ │     ╱ ├───────────────────┼───────────────────────┤
    │ name              │ String                │ └┐   ╱  │ name              │ String                │
    └───────────────────┴───────────────────────┘  │  ╱   └───────────────────┴───────────────────────┘
                                                   │ ╱   
                                                   │╱

**Использование:** В методах загрузки данных для информационного логирования.

NativeDumperValueError
======================

.. py:exception:: NativeDumperValueError

   **Родитель:** ``ValueError``

   Ошибки валидации входных параметров.

   **Когда возникает:**
   - Пустые или некорректные параметры методов
   - Отсутствие обязательных аргументов
   - Некорректные значения параметров
   - Несовместимые комбинации параметров

   **Примеры:**
   - Пустое имя таблицы
   - Отсутствие файлового объекта
   - Некорректный метод сжатия

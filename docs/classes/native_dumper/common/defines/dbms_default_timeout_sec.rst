DBMS_DEFAULT_TIMEOUT_SEC
========================

.. py:data:: DBMS_DEFAULT_TIMEOUT_SEC

   Стандартное время ожидания ответа от сервера ClickHouse в секундах.

   :Тип: int
   :Значение: 300
   :Единица измерения: секунды
   :Эквивалент: 5 минут

**Описание:**

Константа определяет стандартный таймаут для операций с ClickHouse сервером.
Используется при установке соединения и выполнении запросов к ClickHouse.

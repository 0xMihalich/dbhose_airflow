CHUNK_SIZE
==========

.. py:data:: CHUNK_SIZE

   Средний размер чанка для разделения передаваемого объекта в потоке.

   :Тип: int
   :Значение: 16384
   :Единица измерения: байт

**Описание:**

Оптимальный размер чанка, используется как для чтения так и для записи данных при работе с Clickhouse сервером.

pyo3http
========

Модуль на rust для работы с Clickhouse через HTTP/HTTPS интерфейс.

**Импорт из native_dumper**

.. code-block:: python

    # Весь модуль
    from native_dumper.common import pyo3http

    # Сами классы
    from native_dumper.common.pyo3http import (
        HttpSession,
        HttpResponse,
    )

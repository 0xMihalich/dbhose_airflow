HttpResponse
============

.. py:class:: HttpResponse()

   HTTP-ответ, реализующий интерфейс файлового объекта.

**Принцип работы:**

Обертка над HTTP-ответом, написанная на Rust, предоставляющая Python-интерфейс
для чтения данных как из файла. Использует внутренние буферы для эффективной
работы с потоковыми данными из сети.

**Ключевые методы:**

- ``read(size)`` - чтение данных порциями
- ``read1()`` - чтение одного байта
- Методы проверки статуса: ``is_success()``, ``is_error()`` и др.
- Методы работы с заголовками: ``get_headers()``, ``get_header()``
- Методы управления потоком: ``seek(0)``, ``tell()``, ``close()``

**Особенности реализации на Rust:**

* Низкоуровневая работа с сокетами
* Буферизация для минимизации системных вызовов
* Поддержка chunked transfer encoding
* Автоматическое распознавание кодировок
* Эффективное управление памятью (zero-copy где возможно)

**Использование:** Внутри ``HTTPCursor`` для обработки ответов от ClickHouse.

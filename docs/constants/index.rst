DBMS_DEFAULT_TIMEOUT_SEC
------------------------

.. py:data:: DBMS_DEFAULT_TIMEOUT_SEC

   Стандартное время ожидания ответа от сервера ClickHouse в секундах.

   :Тип: int
   :Значение: 300
   :Единица измерения: секунды
   :Эквивалент: 5 минут

**Описание:**

Константа определяет стандартный таймаут для операций с ClickHouse сервером.
Используется при установке соединения и выполнении запросов к ClickHouse.

**Использование:**

.. code-block:: python
   
   from dbhose_airflow import DBMS_DEFAULT_TIMEOUT_SEC

.. note::

    По умолчанию данная константа уже указана в атрибуте DBHose.timeout, но применяется только при работе с Clickhouse.

    Если запрос может выполняться значительно дольше, рекомендуется при инициализации DBHose явно указать время в секундах.
